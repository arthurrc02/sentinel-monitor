# Sentinel Frontend

Dashboard web para visualização em tempo real das métricas coletadas pelo Sentinel.

> Status: scaffolding inicial. Consumo da API e visualizações de métricas ainda serão implementados em sprints futuras.

## Requisitos

- Node.js 20+

## Instalação

```bash
npm install
```

## Execução

```bash
npm run dev
```

A aplicação estará disponível em `http://localhost:5173`.

## Lint

```bash
npm run lint
```
